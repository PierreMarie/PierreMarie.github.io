## Avant de nous écrire (résolution rapide)

1. Vérifiez que **l’app est à jour** depuis l’App Store.  
2. Redémarrez l’app / l’appareil.  
3. Si vous avez un **abonnement**, gérez-le ici : <https://apps.apple.com/account/subscriptions>.  
4. Si le problème persiste, contactez‑nous avec les informations ci‑dessous.

## Pour signaler un bug (checklist utile)

Incluez autant que possible :

- **Version de l’app** (ex. 1.0) — visible dans Réglages → poneyQuiPue.  
- **Appareil & iOS** (ex. iPhone 14, iOS 20.x).  
- **Étapes pour reproduire** (1, 2, 3).  
- **Résultat attendu** vs **observé**.  
- **Captures d’écran / vidéo** (si possible).  
- (Optionnel) **Connexion** : type de compte, état réseau (Wi‑Fi/4G), région.

## FAQ (rapide)

**L’app plante au lancement**  
➡️ Supprimez puis réinstallez l’app depuis l’App Store. Si ça continue, écrivez‑nous avec le modèle d’iPhone et la version iOS.

**Problème d’abonnement / facturation**  
➡️ Gérez ou annulez depuis Apple : <https://apps.apple.com/account/subscriptions>. Pour des achats récents, vous pouvez **signaler un problème** via Apple : <https://reportaproblem.apple.com/>.

**Questions confidentialité & données**  
➡️ Consultez notre politique : <https://pierremarie.github.io/privacy/>.
